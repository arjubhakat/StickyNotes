About the Context Menu

This menu contains shortcuts to open Sticky Notes, as well as allowing you to add links to new notes without having to switch tabs.

Right-click (or CTRL-click if you're on a Mac) anywhere on the page to bring up the context menu. There are always two options: "Open Sticky Notes" and "Save Page."

"Open Sticky Notes" will, of course, open the app. If you've already got Sticky Notes open, clicking this will switch to it. "Save page" adds the current page's URL to a new note.

If you right-click a link, the option "Save link" will save the link's URL to a new note. If you right-click a text selection, "Save selection" will save the selected text to a new note. If you right-click an image, "Save image" will save the image to a new note.

Heads up! Currently, the image itself will be put into a note. Let me know, either on the support page or on the Chrome Store's app feedback page if you would like an option to save the image's path instead.

Please note (no pun intended) that after you enable the context menu option, it will only become active after you restart your browser.
